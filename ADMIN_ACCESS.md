# Admin Panel Access Guide

## Hidden Admin Panel

The admin panel is now hidden from the regular blog view to keep the interface clean for readers.

### How to Access the Admin Panel

**Use Keyboard Shortcut:**
- **Windows/Linux:** Press `Ctrl + Shift + A`
- **Mac:** Press `Cmd + Shift + A`

This will instantly take you to the admin panel where you can:
- Create new posts
- Edit existing posts
- Delete posts
- Export/Import posts
- Preview posts before publishing

### Direct URL Access

You can also access the admin panel directly by typing in your browser:

```
https://yoursite.github.io/admin.html
```

Or locally:
```
http://localhost:8000/admin.html
```

### Tips

- The keyboard shortcut works on all pages (Blog, About, Post view)
- Admin panel is fully functional - only the visible link is hidden
- Regular readers won't see the admin option
- Your blog looks clean and professional for visitors

### Share Admin Access

If you want to share admin access with others:
1. Share your repository on GitHub
2. Give collaborators access
3. They can access admin.html directly or use the Ctrl+Shift+A shortcut

### Bookmark for Quick Access

If you prefer not to use keyboard shortcuts:
1. Go to `admin.html`
2. Bookmark the page (Ctrl+D or Cmd+D)
3. Click bookmark whenever you need to manage posts
