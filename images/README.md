# Images folder

This folder is for organizing uploaded images.

Images uploaded through the admin panel are automatically embedded in post data (as base64), so they don't need to be stored as separate files here.

However, if you want to manually add images:
1. Place image files in this folder
2. Reference them in your posts using markdown: `![alt text](./images/filename.png)`

Supported formats: .jpg, .jpeg, .png, .gif, .webp
